
##DISCLAIMER

Distributes as is. There may be problems with windows handling.
Works fine on Win7 x64, MSVC 2013, Wacom Intuos4 PTK-840.

Two functions only: initWintab() and cleanupWintab().

Access to data through getWT struct.
